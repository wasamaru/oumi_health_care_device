## 備考

### 主な変更点

1. **CN1: microUSB → USB Type-C（5077CR-16-SMC2-BK-TR）に変更**
   - 既存: 10118194-0001LF（Amphenol FCI、廃番）
   - 新規: 5077CR-16-SMC2-BK-TR（Neltron、USB 2.0対応16ピン）

2. **CN4-CN7: DCジャック → USB Type-C（5077CR-16-SMC2-BK-TR）に変更**
   - 既存: PJ1-023（CUI Devices、DCジャック）
   - 新規: 5077CR-16-SMC2-BK-TR（Neltron、USB Type-C）×4個

3. **U1: EYSHCNZXZ（廃番）→ MBN52832（村田製作所）に変更**
   - 既存: EYSHCNZXZ（太陽誘電、廃番）
   - 新規: MBN52832 / WSM-BL241-ADA-008（村田製作所）
   - 同一SoC（nRF52832）のためファームウェア移植が容易

4. **Type-C用CC端子プルダウン抵抗（5.1kΩ×10）追加**
   - USB Type-Cコネクタ5個×2本（CC1, CC2）= 10個
   - 充電開始に必須の回路

5. **基板: 4層基板 @¥1,500**

### 選定理由

#### MBN52832（村田製作所）を選定した理由

1. **開発リスクが最小**
   - 既存EYSHCNZXZ（廃番）と同一のnRF52832 SoCを採用
   - ファームウェアがほぼそのまま使える
   - Nordic nRF5 SDKをそのまま使用可能

2. **筐体薄型化に最適**
   - サイズ: 7.4×7.0×**0.9mm**（厚さ0.9mmは全候補中最薄）
   - アップグレード計画の「薄型化・スリム化」要求に完全合致

3. **技適取得済み**
   - MIC (Japan) Certified取得済み
   - モジュール認証を活用した製品認証が可能
   - 認証コスト不要

4. **高い信頼性**
   - 村田製作所の実績と長期供給保証

#### USB Type-Cコネクタを選定した理由

1. **既存microUSBと完全互換**
   - USB 2.0対応（16ピン）で既存と同等の機能
   - ファームウェア修正が最小限

2. **電源供給能力が向上**
   - 既存microUSB: 最大2A程度
   - Type-C: 最大5A（約2.5倍）

3. **ユーザビリティ向上**
   - リバーシブル（上下どちらでも挿入可能）
   - 使い勝手が大幅に改善

4. **コストパフォーマンス**
   - 価格¥110と非常に安価
   - 秋月電子で即納可能（在庫7223個）

### 技術的注意事項

1. **USB Type-C用回路の追加が必要**
   - CC1, CC2端子に各5.1kΩのプルダウン抵抗が必須
   - ESD保護回路の追加を推奨
   - デカップリングコンデンサ（100nF + 10μF）の追加を推奨

2. **基板設計の変更**
   - USB Type-Cコネクタのフットプリント変更
   - CC端子用プルダウン抵抗の配置
   - 差動ペア配線（D+, D-はインピーダンス90Ω）

3. **ファームウェアの移植**
   - 同一SoC（nRF52832）のため移植は容易
   - 基本的にUSB通信部分は変更不要
   - 開発工数: 約1-2日
